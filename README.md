# MoodMate AI Chatbot 🤖

## 📌 Project Description
MoodMate AI Chatbot is a Python-based chatbot that interacts with users and responds according to their mood and messages. It can recognize emotions such as happiness, sadness, stress, and anger, provide motivational quotes, and engage in simple conversations.

## ✨ Features
- Greets users by name
- Detects moods such as happy, sad, angry, and stressed
- Provides motivational quotes
- Responds to study and exam-related questions
- Tells programming jokes
- Supports continuous conversation until the user exits
- Generates random responses for better interaction

## 🛠️ Technologies Used
- Python
- Random Module
- Conditional Statements
- Loops
- User Input and Output

## 🚀 How to Run the Project
1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python chatbot.py
```

## 💬 Example Conversation

```text
Enter your name: Radhika

Hello Radhika! I am MoodMate AI.

Radhika: I am happy
MoodMate: That's wonderful! Keep smiling.

Radhika: Motivate me
MoodMate: Believe in yourself. Every small step brings you closer to your goals!

Radhika: I have exams
MoodMate: Stay focused and keep practicing. Your effort will show in your results.

Radhika: exit
MoodMate: Goodbye! Take care and have a great day.
```
